  가능객실수 1

ㅁ 입실시간 안내
입실시간 : 13:00 ~ 19:00
퇴실시간 : 11:00

[상세 내용]
* 상품설명 : 캠핑사이트
* 종류 : 강자갈
* 크기 : 6m X 4m

* 사이트 안내 :
- 가능 : 차박, 루프탑텐트
- 불가능 : 도킹텐트 , 캠핑카, 캠핑트레일러, 카라반 불가

* 사이트옆 주차 가능
* 부대 시설 : 공용 화장실, 개수대, 샤워실, 매점, 카페, 영농체험장, 갤러리, 디자인스쿨

* 사이트 기준 허용 장비 : 텐트 1동, 타프 1대, 차량 1대
※ 야영장 무인출입차단기가 있어서 1사이트 1대의 차량출입등록이 가능합니다. 등록되지 않는 차량에 대해서는 차단기가 열리지 않습니다. 예약시 예약자 추가정보란에 차량번호 정확히 입력 바랍니다. 동반차량은 울미숲 공용주차장에 주차가능합니다.

[인원 안내]
*기준인원 4인 최대 인원 5인
기준인원 초과시 1박당 1인 1만원 추가요금 발생합니다. 옵션에서 선택 가능 합니다

[매너 타임]
* 매너 타임 : 오후 10시~아침 7시
- 음주가무, 고성방가, 차량 이동 금지
- 밤 10시 이후 고성방가나 시끄러운 술자리는 타인에게 불쾌감을 주는 행위이므로 캠핑 에티켓을 지켜주시기 바랍니다

[그 외 안내 사항]
* 방문객 금지합니다
* 순간 전력이 높은 전열 제품 사용 금지 (법적 전력량 600W이하로 제한)

[공용샤워실, 개수대 사용시간]
샤워실, 개수대 모두 오전 7시부터 저녁 9시까지만 사용가능합니다.
